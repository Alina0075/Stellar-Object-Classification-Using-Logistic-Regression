# Stellar-Object-Classification-Using-Logistic-Regression
A machine learning project that classifies celestial objects as stars, galaxies, or quasars using Logistic Regression. The model is trained on astronomical survey data and includes preprocessing, feature scaling, and performance evaluation to demonstrate a complete end-to-end classification pipeline.

Key Features

Data cleaning and feature engineering

Standardization using StandardScaler

Logistic Regression model training

Model evaluation with accuracy, confusion matrix, and classification report

Visualization for exploratory data analysis

Tech Stack
Python • Pandas • NumPy • Scikit-learn • Matplotlib

Goal
To build an efficient and interpretable baseline model for stellar object classification and compare performance on imbalanced astronomical datasets.
